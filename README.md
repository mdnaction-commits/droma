# DRŌMA  
**Real-Time Sonic Dramaturgy**

> A practice-based research system for real-time sonic dramaturgy and performable visual notation.

DRŌMA is a real-time audiovisual system that translates live audio into dynamic visual scores through multiple artistic paradigms inspired by 20th-century visual art (Kandinsky, Klee, Miró and others).

The system explores how computationally generated visual languages can function as performable notation and how live human annotation interacts with algorithmic systems in music performance.

---

## Features

- Real-time audio analysis (FFT, RMS, spectral features)
- Multiple visual engines (Kandinsky, Klee, Miró, Narrative, etc.)
- Live drawing / annotation system
- Audio-reactive generative visuals
- Export pipeline (image / video)

---

## Research Context

DRŌMA operates at the intersection of:

- Music notation technology  
- Sonic visualization  
- Computational aesthetics  
- Human–computer interaction in music  
- Practice-based artistic research
